# 📱 MeatShop Mobile

Aplicativo mobile do projeto **MeatShop**, desenvolvido para permitir que clientes realizem compras de produtos de um açougue de forma prática, segura e intuitiva. O app possibilita visualizar produtos, adicionar itens ao carrinho, realizar pedidos e acompanhar o status das compras em tempo real.

---

## 🧾 Visão Geral

O **MeatShop Mobile** é a interface do cliente final, responsável por consumir a API do sistema MeatShop e fornecer uma experiência de compra otimizada em dispositivos móveis (Android e iOS).

Principais objetivos:
- Facilitar o processo de compra de carnes e produtos relacionados
- Reduzir filas e tempo de atendimento no açougue
- Integrar-se ao sistema web administrativo do açougue

---

## 📲 Funcionalidades do Aplicativo

### 👤 Autenticação
- Cadastro de clientes
- Login com e-mail e senha
- Autenticação segura via JWT
- Persistência de sessão

### 🥩 Catálogo de Produtos
- Listagem de produtos disponíveis
- Visualização de detalhes (nome, preço, descrição e imagem)
- Filtro por categoria

### 🛒 Carrinho de Compras
- Adição e remoção de produtos
- Atualização de quantidades
- Cálculo automático do valor total

### 📦 Pedidos
- Finalização de pedido
- Envio do pedido para o backend
- Visualização do histórico de pedidos
- Acompanhamento do status (em preparo, pronto, entregue)

### 🔔 Notificações
- Atualizações de status do pedido
- Promoções e avisos do açougue
